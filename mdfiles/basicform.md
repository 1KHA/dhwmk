Participant Registration Form Requirements
Form Structure:

The registration form will be used to register participant teams.

It must include Team Leader Information:

Full Name

Phone Number

Email Address

Specialty

A field to specify the Number of Team Members (excluding the leader):

Minimum: 2 members

Maximum: 5 members

This range may be updated by the admin in the future.

Dynamic Member Fields:

Based on the number of members selected, corresponding input fields must appear dynamically for each member.

For each member, the following information is required:

Full Name

Phone Number

Email Address

Specialty

Additional Team Information:

Team Name

Team Idea

(Additional fields may be added later.)

Form Submission:

Upon successful submission, the user should receive a "Form submitted successfully" confirmation message.

The submission data should be made available to the admin for review.

Admin Review and Account Creation:

Once an admin approves a submission:

Participant accounts will be automatically created for all team members, including the team leader.

Username for each account will be the participant’s email address.

Password will be set to the last four digits of the participant's phone number.

Login and Dashboard Access:

Participants will be able to log in using their email and phone number.

After login, users will be redirected to the Participant Dashboard.
